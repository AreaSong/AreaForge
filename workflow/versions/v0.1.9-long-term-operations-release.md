## 风险

- 影响：候选阶段仅本地仓库与 GitHub 分支/PR；确认点 1 起影响公开 Release 资产；确认点 2 起影响生产。

- 验证：见 Planning Gate 验证条目。
- 回滚：见 Planning Gate 回滚条目；生产回滚目标固定为 `v0.1.7` 签名 Release 的不可变 digest。
