# 任务计划

## 目标

任务计划用于管理每天应该完成什么，并和计时、大纲、笔记、错题、复盘关联。

## 字段

- 科目。
- 任务标题。
- 任务类型。
- 预计时长。
- 实际时长。
- 优先级。
- 完成状态。
- 所属大纲节点。
- 完成后的复盘记录。

## 初始科目

- 数学。
- 英语。
- 408 数据结构。
- 408 计算机组成原理。
- 408 操作系统。
- 408 计算机网络。
- 政治。

## 状态

- 待开始。
- 进行中。
- 已完成。
- 已延期。
- 已放弃。

## 规则

- 未完成任务进入任务债务系统。
- 完成任务后可以触发鞭策文案。
- 任务可以关联到学习计时。
- 任务可以关联到大纲节点和笔记。

当前今日任务表单已支持选择任务类型，并把类型写入已有 `StudyTask.type` 字段；任务卡片会展示类型、状态和预计时长。Package B Batch 2 已新增拆小任务父子关系和 `TaskDebtEvent` 债务事件账本；Package D Batch D2 已完成重排建议确认、驳回和用户所选项应用记录。批量任务调整或自动阶段联动仍需单独确认。
