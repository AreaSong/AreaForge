# 动机封存与唤醒

## 目标

保存开始的原因，并在关键节点唤醒；另提供克制的动机内容库与「我学不下去了」恢复入口。

## 封存内容

- 为什么开始。
- 最不想回到什么状态。
- 想成为怎样的人。
- 给未来自己的话。
- 第一次全真自测后的阶段日记。

入口：`/settings/profile`（legacy `/motivation` 重定向）；封存正文默认不进入 AI。

## 内容库（Batch 9）

- 类型：`QUOTE` / `VIDEO_LINK`（HTTPS）/ `VAULT_EXCERPT`（用户显式摘录）。
- `POST /api/motivation/next` 返回一条可展示内容与继续 / 5 分钟 / 最小任务三动作。
- `MotivationReminderState` 做 4 小时 / 每日两次频控，跨工作区共享额度。

## 唤醒时机

- 连续失守。
- 重大复盘。
- 第一次全真自测前后。
- 用户主动查看。
- 系统判断进入危险期。

## 原则

- 不让过去的人每天出现。
- 不把伤口做成产品卖点。
- 只在需要重新确认方向时唤醒底层理由。
- 内容消费不是完成指标，不替代真实学习行动。
