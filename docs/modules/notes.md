# 笔记与资料库

## 目标

保存自己的理解、图片、PDF、拍照笔记和阶段材料，并和大纲、任务、错题关联。

## 资料类型

- 文本笔记。
- 图片。
- PDF。
- 拍照上传。
- 课程截图。
- 公式整理。
- 代码片段。
- 题解。

## 字段

- 标题。
- 所属科目。
- 关联大纲节点。
- 关联任务或错题。
- 自己的理解。
- 附件。
- 掌握状态。
- 复习提醒时间。

## 掌握状态

- 理解了。
- 似懂非懂。
- 不会。
- 需要重学。
- 考前再看。

## 规则

- 上传文件不等于掌握。
- 每份资料必须能说明属于哪里、解决什么问题、下次何时再看。
- 文件本体由 `packages/storage` 管理，数据库只保存 metadata。

## 第二阶段增强

- 笔记复习提醒。
- 根据大纲节点和错题关系推荐复习资料。
- 标记考前再看资料。
- 支持按科目、节点、掌握状态筛选。

当前 `/notes` 已支持按科目、考纲节点、掌握状态和复习提醒状态筛选文本笔记。Package A 后，`/notes` 也支持给笔记上传 PDF、PNG、JPEG、WebP 附件，文件写入私有 `UPLOAD_DIR`，列表只展示 metadata 和鉴权下载入口，不暴露内部 URI 或磁盘路径。附件删除、错题/模拟/阶段附件和 AI 解析仍需后续单独确认。
