# 今日作战台

## 目标

今日作战台是每天打开 AreaForge 的第一屏，目标是让用户立刻知道今天要做什么、当前风险是什么、下一步动作是什么。

## 展示内容

- 今日日期。
- 2026 全真自测倒计时。
- 2027 终局考试倒计时。
- 今日学习总时长。
- 有效学习时长。
- 今日任务完成率。
- 当前连续打卡天数。
- 当前阶段名称。
- 当前风险等级。
- 今日鞭策文案。
- 快速开始学习计时。

## 信息优先级

第一优先级：

- 当前推荐任务。
- 开始计时。
- 风险等级。

第二优先级：

- 今日指标。
- 欠账任务。
- 考纲进度概览。

第三优先级：

- 错题提醒。
- 复盘提醒。
- 阶段称号。

## 状态影响

- 正常推进：展示完整作战台。
- 连续高完成：突出连胜和阶段突破。
- 断签警报：突出欠账和恢复动作。
- 恢复模式：只展示最小任务。
- 冲刺模式：前置真题、模拟、错题。
