# 鞭策引擎

## 目标

根据真实进度动态生成反馈，而不是固定鸡汤或固定毒舌。

## 语气类型

- 温和鼓励型。
- 严肃教练型。
- 毒舌逼迫型。
- 数据冷冰冰型。

## 状态类型

- 上升期：连续学习、完成率高、进度健康。
- 稳定期：持续学习，但增长一般。
- 波动期：有学习，但经常拖延或质量不稳。
- 失守期：连续多天未打卡或任务大量失败。
- 危险期：长期落后，计划明显失效。
- 冲刺期：临近考试，优先真题、模拟、错题复盘。

## 原则

- 学得好时，不灌鸡汤，继续加压。
- 学得差时，不只批评，必须给最低可执行动作。
- 长期断签时，进入恢复模式，先恢复行动感。
- 狠，但不摧毁。
