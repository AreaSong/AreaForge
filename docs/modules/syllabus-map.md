# 考纲进度树与作战地图

## 目标

把“学了多久”进一步落到“学到了哪里”。

## 考纲树

每个科目有一棵大纲树。

节点层级：

- 科目。
- 章节。
- 知识点。
- 题型或专题。

节点状态：

- 未开始。
- 学习中。
- 已学完。
- 需要复习。
- 掌握。
- 薄弱。
- 放弃或暂缓。

节点标记：

- 打勾：已完成或已掌握。
- 打叉：当前不会、理解失败或需要重学。
- 星标：高频重点或个人重点。
- 警告：长期未复习或错题集中。

## 作战地图

作战地图是所有科目进度的可视化概览。

格子状态：

- 未开始。
- 学习中。
- 已覆盖。
- 已验证。
- 薄弱。
- 遗忘风险。
- 错题高发。

第一版先做树形进度 + 网格概览。

## 第二阶段增强

- 标记长期未复习节点。
- 根据错题集中度生成遗忘风险。
- 根据计时、任务、错题、复盘自动更新节点状态。
- 作战地图支持更强的视觉表达和筛选。
