# AreaForge Charter

## 产品定位

AreaForge 是一个面向个人长期备考的自我锻造与考研督战系统。

它不是普通打卡软件，也不是单纯待办清单，而是围绕一个长期目标持续追踪学习投入、任务完成、知识掌握、错题复盘、状态波动和阶段风险的私人作战台。

## 当前目标

- 考试方向：考研计算机，公共课 + 408。
- 目标院校：清华。
- 目标分数：450。
- 当前基础：零基础。
- 起始日期：2026-07-05。
- 第一节点：2026 年 12 月下旬，和 27 考研同步参加一次全真自测。
- 终局节点：2027 年 12 月下旬，作为真正上岸目标。
- 使用设备：电脑为主，手机补充访问。

## 命名

产品名：AreaForge。

- `Area` 延续 AreaFlow、AreaMatrix 产品体系。
- `Forge` 表示锻造、淬炼，把当前的自己打造成更强的自己。

中文气质关键词：

- 锻造
- 督战
- 复盘
- 压强
- 长期主义
- 冷静但不麻木

## 核心原则

1. 先让用户每天回来。
2. 数据必须推动行动。
3. 鞭策必须和状态相关。
4. 狠但不摧毁。
5. 数据和代码分离。

## 核心闭环

```text
计划任务 -> 专注计时 -> 关联大纲 -> 产出笔记/错题 -> 晚间复盘 -> AI 鞭策 -> 明日调整 -> 周期报告
```

## 第一版边界

- 私有 Web 应用。
- 电脑端优先。
- 响应式适配手机。
- 数据部署在自己的服务器。
- AI 只做建议和表达，不直接改写用户记录。

第一版不追求公开平台、多用户排名、小程序或原生 App。
