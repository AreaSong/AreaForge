# 功能范围

## 第一版必须有

- 单管理员登录。
- 今日作战台。
- 双节点倒计时。
- 每日任务。
- 任务债务基础版。
- 学习计时。
- 专注计时模式。
- 打卡。
- 每晚复盘。
- 考纲进度树。
- 知识点掌握状态。
- 知识点掌握证明基础版。
- 笔记与资料上传。
- 情绪与状态记录基础版。
- 恢复模式基础版。
- 反假学习检查基础版。
- 考研作战地图概览版。
- 动机封存。
- 阶段称号基础版。
- 鞭策文案。
- AI 复盘建议。
- AI 明日任务建议。
- 基础统计。
- 数据持久化。

## 第一版简化实现

- 作战地图先用树形进度 + 网格概览。
- 任务债务先支持补做、延期、放弃。
- 恢复模式先通过规则触发。
- 反假学习先通过结束计时后的问题检查。
- 知识点掌握证明先做手动条件勾选。
- 情绪记录先做标签和备注。
- 阶段称号先按连续打卡、时长、任务完成率计算。

## 第二阶段增强

- 全真模拟考试模式完整实现。
- 2026 年 12 月同步自测专题流程。
- 周审判报告。
- 月复盘报告。
- 任务债务自动重排建议。
- 知识点遗忘风险提醒。
- 笔记复习提醒。
- 作战地图高级可视化。
- 状态主题深度联动。
- 动机唤醒机制。
- AI 根据长期数据生成阶段调整建议。

## 下一产品版本规划（隔离实现，生产未切换）

以下能力已在隔离分支实施，但尚未通过完整 minor Release admission 或生产切换；不得计入当前生产或 docs 100% 完成声明。权威规格见 `workflow/versions/v1.1-learning-action-center.md`；实现状态见 `docs/development/feature-traceability.md`。

- 五个稳定工作台与受保护应用壳（今日、知识、复盘、阶段、设置）。
- 当前考试工作区、自定义科目、408 分组与历史工作区归档。
- 今日行动中心：科目快捷计时、单一推荐、三队列、空态最小任务。
- 计划收件箱、任务软/硬依赖、滚动七天计划与欠账区。
- 统一复习（排期 + 事件）、快速复习、CheckIn 快照演进与恢复三阶。
- 学习树模板下载、规范化 Markdown 预览/原子确认、导入历史与一次性导出。
- 全局知识关联画布（派生关系、分层加载、个人布局）。
- 六类知识卡片、资料资产（FILE/LINK）、动机内容库与克制提醒。
- 浏览器前台通知（默认隐藏具体标题）与四类显式 AI 文本草稿。
- 模拟分科总分、结构化失分、报告/阶段原子入箱边界。

本规划明确不包含：内置 OCR、后台 Push、自动应用完整学习计划、自由白板、视频托管、物理删除资料/导入历史、多用户与社交。
