# 备份与恢复

## 备份对象

- PostgreSQL 数据库。
- 上传文件目录。
- 生产 `.env`。
- 当前部署版本 tag。

## 策略

- 每天自动 `pg_dump`。
- 至少保留 14 天。
- 上传目录与数据库同周期备份。
- 正式发布前生成手动备份点。

## 恢复验证

备份必须能恢复到临时库并启动应用，才算有效。

恢复演练至少验证：

- 数据库可导入。
- 附件 metadata 和文件本体能对应。
- 首页能读取数据。
- 登录仍可用。

## 回滚

- 保留上一个 Docker 镜像 tag。
- migration 涉及破坏性字段时先写回滚说明。
- 第一版避免不可逆 migration。
