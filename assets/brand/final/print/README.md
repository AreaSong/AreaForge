# AreaForge Print Assets

## 文件选择

- `*.svg`：轮廓化矢量源，适合排版、缩放和专业设计工具。
- `*.pdf`：可直接预览和交换的单页矢量文件；dark 版本包含品牌深色背景，保证反白字标可见。
- `*-cmyk.tiff`：`3600x1170`、300 DPI、LZW 压缩的 CMYK 位图，用于不接收 SVG/PDF 的印刷流程。

CMYK 文件是从 sRGB 品牌色转换的通用起点，正式大批量印刷仍需由印厂按纸张、油墨和设备打样。

本包不提供 EPS。EPS 无法可靠保留当前 Logo 的透明度、渐变和阴影，并且当前 macOS 已不能稳定完成渲染回读验证。需要旧版设计软件兼容时，应优先导入 outlined SVG 或矢量 PDF，再由目标软件按自身色彩配置导出。
